EV Control Unit (CAN-Based)

This article explains the design of a CAN-based main control unit for a low-speed electric vehicle, focusing on system architecture and control constraints.

System Overview

The control unit coordinates throttle input, motor drive, direction control, and display feedback.

Hardware Architecture

  • STM32 microcontroller
  • MCP2515 CAN controller
  • 48 V, 3 kW DC motor

CAN Message Design


        ID: 0x101
        Byte0: Throttle
        Byte1: Direction

Results

The system achieved stable torque control and reliable communication under load.
